## Navigation Flow

### From Landing Page:
1. **About Section Cards** (4 cards) → Click "Explore Program" → Navigate to respective program detail page
2. **Back to Home Button** → Returns to landing page from any program detail page
3. **Programs Section** → Age-based groupings with links to relevant programs

### Design Consistency:
- All program pages use the same structure and layout patterns
- Color schemes match the program:
  - Blue for Coding
  - Purple for Math  
  - Green for Abacus
  - Orange/Pink gradient for UX/UI Design
- Consistent typography (Clash Grotesk for headings, Inter Tight for body)
- Unified spacing system (py-16 for sections)
- Responsive design throughout

## Mobile Responsiveness:
- Grid layouts adapt from desktop (2-4 columns) to mobile (1 column)
- Typography scales appropriately
- Images maintain aspect ratios
- Buttons stack vertically on small screens
- Navigation simplified for mobile
